How to Use This Bookmark as a Dashboard Tile

  3. Create a new shortcut on your desktop (right-click > New > Shortcut)

  5. Name it "Ketamine Question Bank" and click Finish

  7. Choose an appropriate icon
  8. Pin this shortcut to your Start Menu or Taskbar
