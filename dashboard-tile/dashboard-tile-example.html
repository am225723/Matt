How to Add This Tile to Your Dashboard

There are several ways to add a clickable tile to your dashboard, depending on what type of dashboard you're using:

Option 1: Bookmark Tile

Create a bookmark tile that opens the Ketamine Question Bank:

  1. Host the Ketamine Question Bank on a web server or cloud storage service (like GitHub Pages, Netlify, or AWS S3)
  2. Create a bookmark to that URL in your browser
  3. For browsers or operating systems that support bookmark tiles (like Chrome OS or Windows Start Menu), pin this bookmark as a tile

Option 2: HTML Dashboard

If you have an HTML dashboard, add this code to create a tile:

<div class="dashboard-tile ketamine-tile" onclick="window.open('https://your-hosted-url.com/ketamine-question-bank.html', '_blank')"> <div class="tile-header">Ketamine Question Bank</div> <div class="tile-content"> <div class="tile-icon"><i class="fas fa-brain"></i></div> <div class="tile-description">Reflect and record your thoughts</div> </div> </div>

Option 3: Data URI Approach

Create a bookmark that contains the entire application using a data URI:

  1. Convert the entire Ketamine Question Bank HTML file to a data URI
  2. Create a bookmark with this data URI as the URL
  3. Pin this bookmark to your dashboard or start menu

Here's a sample bookmark URL (shortened for demonstration):

javascript:(function(){ window.open('data:text/html;charset=utf-8,' + encodeURIComponent('Ketamine Question Bank '), '_blank'); })();
Note: The actual data URI would contain the entire Ketamine Question Bank HTML code, which is too large to show here. You can generate this by using an online tool to convert the HTML file to a data URI.

Option 4: Web Application Shortcut

Modern browsers allow you to create web application shortcuts:

  1. Host the Ketamine Question Bank on a web server
  2. Open the page in Chrome or Edge
  3. Click the three dots menu → More tools → Create shortcut
  4. Check "Open as window" for a more app-like experience
  5. The shortcut will appear on your desktop or start menu

Option 5: iframe Integration

If your dashboard supports iframe embedding, use this code:

<iframe src="https://your-hosted-url.com/ketamine-question-bank.html" title="Ketamine Question Bank" style="width: 100%; height: 600px; border: none; border-radius: 8px; box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);"> </iframe>

Hosting Options for the Ketamine Question Bank

To make the application accessible via a URL (needed for options 1, 2, 4, and 5), you'll need to host it somewhere:

Free Hosting Options:

  • GitHub Pages: If you have a GitHub account, you can host the HTML file for free
  • Netlify: Offers free hosting for static sites with drag-and-drop deployment
  • Vercel: Similar to Netlify, with free hosting for personal projects
  • AWS S3: Low-cost storage that can host static websites (has a free tier)
  • Google Drive: You can share HTML files publicly, though with some limitations

Steps for GitHub Pages Hosting:

  1. Create a GitHub repository (or use an existing one)
  2. Upload the Ketamine Question Bank HTML file
  3. Go to repository Settings → Pages
  4. Select the branch to deploy (usually main) and save
  5. Your site will be available at https://[username].github.io/[repository-name]/ketamine-question-bank.html
